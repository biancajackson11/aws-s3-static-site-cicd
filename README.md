# 🚀 Production-Ready Static Website on AWS (CI/CD)

## 📌 Overview

This project demonstrates a fully automated CI/CD pipeline deploying a static website to AWS.

## 🏗️ Architecture

* Amazon S3 (static website hosting)
* GitHub Actions (CI/CD pipeline)
* AWS IAM (secure deployment permissions)

## ⚙️ CI/CD Pipeline

* Code pushed to GitHub triggers GitHub Actions
* Pipeline builds and deploys automatically to S3
* No manual uploads required

## 🔐 Security

* IAM roles configured for least privilege
* Secure deployment via GitHub secrets

## 📊 What I Learned

* How to automate deployments using CI/CD
* AWS S3 hosting in a production-like setup
* Managing credentials securely with IAM
